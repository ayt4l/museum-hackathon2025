# <b>Проект команды НУГ-1</b>
## Кейс: Музей

Веб-сайт виртуального музея с использованием 360 панорам.
Сайт разработан для Республиканскому музею истории государственности РС(Я) имени Максима Кировича Аммосова в рамках муниципального этапа хакатона "Моя Профессия - ИТ" с 10 по 12 декабря 2025 года.
